ROIB - Remove [facebook element] on IPS Block

Removes an element [Facebook iframe] when Facebook is blocked by ISP. This will remove annoying ISP block page from your webpage and reduce page load time when Facebook is blocked.

How to use

Add roib.min.js within <body></body>. See example.html for more info.
